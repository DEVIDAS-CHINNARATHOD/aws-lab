## ☁️ Cloud Computing Lab – BCS601

This repository contains the **Cloud Computing Lab programs and solutions** for **BCS601 (6th Semester CSE – VTU 2024-25 Scheme)**.

The goal of this repository is to document all the **lab experiments, commands, and implementations** related to cloud platforms like **Google Cloud Platform (GCP)** and related services.

Each experiment includes:
- Objective
- Steps / Commands
- Implementation
- Output 

These programs are based on the **Cloud Computing Lab Manual** provided by the department.
